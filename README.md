# letter
Testing example of a LaTeX project.
It is intended for studying and practicing with Git.

1.- signing tag: $> git tag -s v1.0 -m 'BTA: This is my signed 1.0
tag. From Europa'

Show the tag:
2.- $> git show v1.0

https://git-scm.com/book/en/v2/Git-Tools-Signing-Your-Work
